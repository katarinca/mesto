# Проект: Место

**Ссылка на сайт:** <https://katarinca.github.io/mesto/>

### Описание
Проектная работа адаптивного сайта с приминением JavaScript

***Используемые технологии:***

* HTML и CSS
* Адаптивная вёрстка
* Grid
* Flexbox
* Файловая структура по БЭМ
* Анимационные эффекты
* JavaScript
